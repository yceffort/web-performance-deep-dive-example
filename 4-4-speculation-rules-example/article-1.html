웹 성능 최적화 기초

prefetch로 로드됨

웹 성능 최적화는 사용자 경험의 핵심입니다. 페이지 로드 속도가 1초 지연될 때마다 전환율은 7% 감소하고, 페이지 뷰는 11% 줄어듭니다.

Core Web Vitals

Google이 정의한 핵심 웹 지표는 세 가지로 구성됩니다:

  • LCP (Largest Contentful Paint): 가장 큰 콘텐츠가 화면에 표시되는 시간
  • INP (Interaction to Next Paint): 사용자 상호작용에 대한 응답 시간
  • CLS (Cumulative Layout Shift): 레이아웃 변경으로 인한 시각적 안정성

측정 도구

성능 측정에는 다양한 도구를 사용할 수 있습니다. Chrome DevTools의 Performance 탭, Lighthouse, WebPageTest 등이 대표적입니다.

이 페이지는 prefetch로 미리 로드되었습니다.
