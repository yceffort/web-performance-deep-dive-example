Paint Flashing & Layer Borders 예제

Chrome DevTools의 Rendering 탭에서 Paint Flashing과 Layer Borders를 켜고 각 요소를 호버해보세요.
페인트가 발생하면 녹색으로 깜빡이고, 레이어가 분리되면 오렌지색 테두리가 표시됩니다.

테스트 방법

  1. DevTools 열기 (F12 또는 Cmd+Option+I)
  2. Cmd+Shift+P → "Show Paint Flashing" 검색 → 활성화
  3. Cmd+Shift+P → "Show Layer Borders" 검색 → 활성화
  4. 각 요소를 호버하며 페인트 발생 여부와 레이어 분리 확인
  5. Cmd+Shift+P → "Show Layers" 검색 → Layers 패널에서 Compositing Reasons 확인
