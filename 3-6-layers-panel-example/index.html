Layers 패널 확인 방법

  1. Chrome DevTools 열기 (F12 또는 Cmd+Option+I)
  2. 우측 상단 점 세 개 메뉴 → More tools → Layers 선택
  3. 또는 Cmd+Shift+P → "Show Layers" 검색
  4. 각 레이어를 클릭하여 Compositing Reasons 확인
